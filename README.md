# Barbershop Agenda

- App runs both in Ios and Android.

- tested in Android 10, 11, 09.

## MAIN LIBS
- [bloc_pattern] - state management
- [get] - used to navigate
- [flutter_form_builder] - forms
- [onesignal_flutter] - push notification integration
- [firebase] - backend
